General Information about this Application

This web application was created within the scope of my (Niklas Furtlehner) bachelor thesis in the bachelors degree of Computer Science at the Johannes Kepler University Linz. The corresponding lecture is PR Projektpraktikum 351.067.

The application extracts courses from the course program of the Red Cross of 2023 and then displays them as a list. The user has the possibility to filter and search the list. The application also displays charts representing the course data and provides a PDF upload option.
